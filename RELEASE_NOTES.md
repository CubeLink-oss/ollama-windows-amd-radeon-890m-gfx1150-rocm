# Release Notes

## v0.25.0-rc0-gfx1150-rocm-windows-2026-05-23

Unofficial Ollama Windows ROCm/HIP build for AMD Radeon 890M Graphics, `gfx1150`.

### Included

- `ollama.exe`
- CPU backend DLLs
- ROCm/HIP backend DLLs
- `ggml-hip.dll`
- rocBLAS runtime files, including `gfx1150` kernels

### Tested

- Windows 11 Pro for Workstations
- AMD Radeon(TM) 890M Graphics
- AMD HIP SDK 6.4.2
- `qwen2.5:3b`
